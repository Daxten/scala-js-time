# Moment.js façade for Scala.js

## Installation
Add the following dependency to your `build.sbt`:

```scala
libraryDependencies +=
  "io.github.widok" %%% "scala-js-momentjs" % "0.1.4"
```

## Example
```scala
import org.widok.moment._

Moment.locale("en_GB")

Moment().calendar()
Moment(millis).calendar()
Moment(millis).fromNow()
```
